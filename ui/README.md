# Available Scripts

In the project directory, you can run:

## Running

- Start API in local machine under [http://localhost:8000](http://localhost:8000)
- Run `npm start`
- App runs under [http://localhost:3000](http://localhost:3000)

## Testing

- Start API in local machine under [http://localhost:8000](http://localhost:8000)
- Run `npm test`

## Debugging

- Start API in local machine under [http://localhost:8000](http://localhost:8000)
- Run `npm start`
- App runs under [http://localhost:3000](http://localhost:3000)
- Use `Attach Chrome` under debug options
